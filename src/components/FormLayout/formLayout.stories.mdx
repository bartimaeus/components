# FormLayout

`FormLayout` is an extension of ant design's [Form](https://ant.design/components/form/#Form-Component) component.

## Usage

You can align the controls of a form using the `layout` propï¼š

- `horizontal` to horizontally align the labels and controls of the fields. (Default)

- `vertical` to vertically align the labels and controls of the fields.

- `inline` to render form fields in one line.

```jsx
import { Form, Field } from 'react-final-form'
import { FormItem, FormLayout } from '@bartimaeus/components'

const Form = props => {
  const [user, updateUser] = useState({})
  const save = values => updateUser(values)

  return (
    <Form
      initialValues={{ first_name: 'Barry', last_name: 'Allen' }}
      onSubmit={save}
      render={({ handleSubmit }) => (
        <FormLayout onSubmit={handleSubmit}>
          <Field
            component={FormItem}
            name="first_name"
            label="First Name"
            type="text"
          />
          <Field
            component={FormItem}
            name="last_name"
            label="Last Name"
            type="text"
          />
          <button onClick={handleSubmit}>Save</button>
        </FormLayout>
      )}
    />
  )
}
```
